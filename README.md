# Turiyasoftech app

React (Vite) marketing site: home, company, services (with detail routes), industries, products, blog, contact.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Repository

This project is intended to live under the **TuriyaSoftech** GitHub organization.

After an empty repository is created on GitHub (see org settings → **Repositories** → **New**), push the `main` branch:

```bash
git remote add origin https://github.com/TuriyaSoftech/<repo-name>.git
git push -u origin main
```

Replace `<repo-name>` with the repository name you create (for example `turiyasoftech-app`).
